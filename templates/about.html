About anniedoc

anniedoc is a prototype for annotating any text-based document inline via the web. It uses Shakespeare's Hamlet, a public-domain text marked up and made available on the web by MIT, as its sample document.

This project attempts to balance encouraging viewer engagement and collaboration with providing a calm, focused, uncluttered reading environment.

Libraries used include Flask, LESS, JQuery, jquery-tmpl, and Bootstrap 3. Source code is available on Github.

anniedoc was built by Karen Rustad as part of DevelopHer 2013.
